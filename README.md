Este repositório apresenta meu primeiro projeto completo de site, desenvolvido com HTML5 e CSS3, com o tema voltado à história e evolução do sistema operacional Android.

O objetivo foi aplicar, na prática, conceitos fundamentais de desenvolvimento web, como estruturação semântica, organização de conteúdo, estilização com CSS e criação de um layout simples e funcional.

Durante o desenvolvimento, trabalhei com elementos como imagens, textos informativos, cores e tipografia, buscando criar uma experiência visual agradável e de fácil leitura.

Este projeto marca o início da minha jornada no desenvolvimento front-end e representa um passo importante na consolidação dos meus conhecimentos.

🚀 Tecnologias utilizadas: HTML5 e CSS3
📚 Foco: Estruturação, estilização e organização de conteúdo
💡 Objetivo: Prática inicial e evolução no desenvolvimento web
